Projects

In addition to my dayjob at Solinor Oy I have various hobby projects. I've said that even if I would work as a garbage collector I would still do web programming as a hobby.

Hobby-projects come and go. Some never see the light of day but are dumped. Here is some worth mentioning:

DaysWithout.info

DaysWithout.info is a day-counter for ... anything. The idea is that anyone can create a counter for anything.

SpeedRead.js

SpeedRead.js is a javascript implementation of the speed reading concept by Spritz. People use up to 80 % of time they use reading to physically move their eyes. Removing need to do this greatly improves the reading speed. Check out the live demo.

WhatsUp?

Social media and domain search tool. Idea was to provide single search-tool for Start-Ups to search multiple available social media nicks and domains. Live here.

Feikki.fi

As in fake (in finnish). Original idea was to provide a fake TUPAS-identification service (provided by finnish banks) for developers who need to use that identification method in their projects. The work is far from done - I might finish it later. The site (not service) is live here.
